Consensus Pipeline

Multi-engine molecular docking with structural consensus scoring
Runs Smina, Vina, and Gnina in parallel against a PDB target. Scores are aggregated, poses ranked by RMSD deviation, and the top-binding conformation is returned.
What you're seeing
  • ▸Smina, Vina, Gnina running simultaneously against PDB: 6Y2E
  • ▸Binding affinity (kcal/mol) aggregated across all three engines
  • ▸RMSD matrix computed — top consensus pose identified
  • ▸ESMFold preprocessing for apo-structure inputs
  • ▸Output downloadable as ranked SDF / PDBQT file set

Smart Switch / Project Forge

Intent-based workflow routing from a single natural language command
A researcher types 'Dock Aspirin to 6Y2E'. Smart Switch parses the biomedical intent, routes to the Consensus Pipeline, scaffolds a project, and begins execution — no form-filling, no configuration.
What you're seeing
  • ▸NLP parser extracts: ligand name, target ID, task type
  • ▸Routing engine maps intent → correct workflow module
  • ▸Project Forge auto-creates job with inputs, outputs, and audit log
  • ▸Supports: docking, primer design, literature mining, ADR detection
  • ▸Fallback disambiguation: ambiguous commands prompt a structured clarification

primerforge_demo.mp4 GENOMICS

VigyanLLM Primer

Thermodynamically validated primer pairs delivered in under 30 seconds
Enter a gene name or NCBI accession. VigyanLLM retrieves the sequence, runs thermodynamic profiling, checks for secondary structures, and returns a validated primer pair — lab-ready.
What you're seeing
  • ▸Accepts: gene symbol, NCBI accession, Ensembl ID, or raw FASTA
  • ▸Outputs: Tm (°C), GC%, amplicon size (bp), ΔG (kcal/mol)
  • ▸Hairpin and primer-dimer risk scores computed via nearest-neighbor model
  • ▸Off-target BLAST check against reference genome
  • ▸Output format: table + downloadable CSV for lab orders
