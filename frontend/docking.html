Input Requirements

Protein Sequence

Standard single-letter amino acid code (A-Z). Minimum 10 residues for ESMFold prediction.

Ligand SMILES

Valid SMILES strings, one per line. Each must be parseable by RDKit. Examples:

CC(=O)Oc1ccccc1C(=O)O (aspirin)
Top N

Number of top Vina hits forwarded to GNINA for CNN re-scoring. Default: 50.

Stage Descriptions

Stage 1: ESMFold

Predicts 3D protein structure from sequence using Meta's ESM-2 transformer model. Runs locally with no external API calls.

Stage 2: Vina

Screens all ligands with AutoDock Vina using the predicted structure. Fast physics-based scoring (kcal/mol, more negative = better).

Stage 3: GNINA

Re-scores top Vina hits with GNINA's CNN model. Consensus rank = 0.4 × Vina + 0.6 × GNINA score.

Interpretation

Vina score < −7 kcal/mol: strong binding. −5 to −7: moderate. > −5: weak.

pLDDT > 90: high confidence. 70–90: medium. < 70: low.

Consensus rank combines both methods for improved reliability.
