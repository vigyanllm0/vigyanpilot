Security

Last updated: 2026-06-09

Production Security Commitments

  • HTTPS-only public deployment.
  • Server-authoritative pricing and quota calculation.
  • Razorpay payment signature verification before crediting runs.
  • Rate limiting for authentication, payment, and API endpoints.
  • Admin tools protected by authenticated admin access.
  • Audit logging for security, payment, and restricted-use events.

Responsible Disclosure

Please report suspected vulnerabilities to contact@vigyanllm.in. Include a clear description, steps to reproduce, and impact. Do not access or modify other users' data.

Payment Security

VigyanLLM does not store card details. Payment credentials are handled by Razorpay checkout and bank/payment-provider systems.
