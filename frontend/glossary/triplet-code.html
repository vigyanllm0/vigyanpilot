triplet code

Molecular Biology

Definition

The triplet code is the genetic coding system where three consecutive nucleotides (codon) specify one amino acid, with 64 codons encoding 20 amino acids and 3 stop signals in a degenerate, nearly universal code.

Detailed Explanation

The genetic code in which each amino acid in a protein is specified by a sequence of three consecutive nucleotides (a codon) in the mRNA molecule. There are 64 possible codons (4 nucleotides x 3 positions) encoding 20 standard amino acids plus 3 stop signals. The code is degenerate (redundant) — most amino acids are encoded by multiple codons — and nearly universal across all organisms.
