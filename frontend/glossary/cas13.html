CRISPR-Cas13

Genome Editing

Definition

Cas13 is a Type VI CRISPR system targeting RNA (not DNA) using crRNA guides, with collateral RNase activity exploited in SHERLOCK diagnostics for sensitive RNA detection, being developed for RNA editing and viral targeting.

Detailed Explanation

A class 2 type VI CRISPR system that targets RNA rather than DNA. Cas13 uses a crRNA guide to cleave single-stranded RNA, providing RNA interference without DNA modification. Cas13 exhibits collateral RNase activity upon target recognition, cleaving nearby non-target RNA molecules — this property is used in SHERLOCK diagnostics for sensitive RNA detection. Cas13 is being developed for RNA editing, transcriptome engineering, and viral RNA targeting.
